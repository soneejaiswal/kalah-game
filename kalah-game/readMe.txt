*********************************************** 6-Stone Kalah Game Implemetationc ********************************************************

Game Rules (Assumptions made) : 
1. 2 players can play the game, Players (First player - bottom, Second player - top ) are maintained implicitly
2. Other game rules to be followed as mentioned for 6-stone kalah game PDF document provided.


Technology stack used : 
1. Java 8
2. Spring Framework & Spring boot 
3. Maven 
4. Junit

How to run :
1. run command  'mvn spring-boot:run' from the commandLine by doing cd to /pathToProject
 
JavaDoc/Documentation : 
1. Swagger - can be accessed with endpoint 'http://localhost:8080/swagger-ui.html', after running the project

Test cases :
1. Junit test cases in src/test/java
